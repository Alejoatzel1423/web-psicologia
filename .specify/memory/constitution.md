# Web Psicología Colombia — Constitution

## Core Principles

### I. Primero Móvil (Mobile-First)
El sitio debe funcionar perfectamente en celulares antes que en desktop. Colombia tiene más del 70% de tráfico web desde móviles.

### II. Enfoque en el Usuario Colombiano
Diseñado para el público colombiano: lenguaje claro, regionalismos neutros, formularios simples, diseño acogedor y profesional. Los servicios ofrecidos son los más solicitados en Colombia.

### III. Contenido Antes que Código
La web es informativa y de captación. Las páginas deben estar optimizadas para contenido (texto, imágenes, videos cortos, testimonios) antes que para funcionalidades complejas.

### IV. Simplicidad y Velocidad
Nada de frameworks pesados. HTML + CSS + JS vanilla o mínimo. Carga rápida en conexiones móviles 4G. Priorizar rendimiento sobre efectos visuales.

### V. SEO y Conversión
Cada página debe estar optimizada para buscadores (Google Colombia) y tener calls-to-action claros para captar leads (WhatsApp, formulario, llamada).

## Stack Tecnológico

- **Frontend**: HTML semántico + CSS (Tailwind o vanilla) + JS vanilla
- **Despliegue**: Static site (GitHub Pages, Netlify, o similar)
- **Formularios**: Servicio externo (Formspree, Google Forms, o backend mínimo)
- **Sin framework JS pesado**: Nada de React, Vue, Angular
- **Sin backend**: Sitio 100% estático

## Estructura del Sitio

1. **Inicio** — Hero + servicios principales + CTA WhatsApp
2. **Servicios** — Consultas, Talleres, Seminarios (cards detalladas)
3. **Sobre Mí** — Perfil del psicólogo, credenciales, enfoque
4. **Blog/Recursos** — Artículos de psicología (SEO)
5. **Contacto** — Formulario + WhatsApp + ubicación

## Contenido por Servicio

### Consultas Psicológicas
- Individual, pareja, familiar
- Presencial y online (videollamada)
- Enfoque: cognitivo-conductual, humanista, según el profesional
- Duración típica: 50-60 min

### Talleres
- Grupos pequeños (8-15 personas)
- Temas: manejo de ansiedad, autoestima, habilidades parentales, inteligencia emocional
- Formato: presencial o virtual
- Duración: 2-4 horas, 1-2 sesiones

### Seminarios
- Conferencias y capacitaciones
- Para empresas e instituciones
- Temas: clima laboral, liderazgo, estrés laboral, salud mental organizacional
- Formato: presencial o virtual
- Duración: medio día o día completo
